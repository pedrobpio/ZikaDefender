This app was developed to create a fun and interactive way to learn how to stay safe and not be infected by the Zika virus.

The idea was to find an engaging way for users to learn about the virus seamlessly. A game seemed to be a good idea.

The game is released in English and Portuguese.

Hope you will have fun playing ZikaDefender!!!!! Enjoy the learning experience!!!!!
